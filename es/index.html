Soy programador de computadoras de Uruguay viviendo en Escocia. Este es mi sitio web personal, pero mi rinconcito principal de internet es mi blog Picando Código donde he estado escribiendo desde 2007.

Programador

Trabajo con computadoras, más que nada escribiendo código. Se puede ver parte del código que he escrito en mi perfil de GitHub y mantengo mi CV en este sitio.

Uso GNU/Linux y Software Libre en lo posible. Podés ayudar al Software Libre uniéndote a la Free Software Foundation. Uso y recomiendo Mozilla Firefox como navegador web y Emacs es mi editor de texto.
